# DRRK2 Portfolio - Premium Dark Luxury Tech

This is a boutique landing page built with a "Dark Luxury Tech" aesthetic.

## Design Specs
- **Backgrounds:** Obsidian scales starting at `#09090B`.
- **Accents:** Titanium Champagne (`#AE9775`).
- **Typography:** 
  - Titles: Plus Jakarta Sans
  - Body: Inter
  - Tech: JetBrains Mono

## Features
- **Scroll Reveal:** Intersection Observer with staggering.
- **Magnetic Cards:** Real-time mouse tracking for light effects.
- **Parallax:** Multi-layer hero movement.
- **Responsive:** Optimized for mobile, tablet, and 4K displays.

## Structure
```
Portafolio/
├── index.html     # Core structure
├── styles.css     # Design system & animations
├── script.js     # Interactivity & reveals
└── images/        # Placeholders (awaiting generation)
```

## Setup
Simply open `index.html` in any modern browser. No build steps required.
